Event attributes

HTML events are "things" that happen to HTML elements. When JavaScript is used in HTML pages, JavaScript can "react" on these events.

BLUR EVENT

Blur event is used when we want the element to resonate when it loses focus.


FOCUS EVENT

Focus event is used when we want the element to resonate when it is in focus.


CONTEXTMENU EVENT

The oncontextmenu event occurs when the user right-clicks on an element to open the context menu.

ON CLICK

If we add onclick attribute to a javascript code , the value of this attribute will be executed if we click on it.


ON MOUSE OVER

When the mouse cursor is on an element, element will change the color

ON MOUSE ENTER

When the cursor mouse enters the space the element occupies , element will change the color

LocalStorage & SessionStorage

LocalStorage and SessionStorage serve to store information about the site in them, store the information in the pairs name = value, you can see the syntax below.
The difference between localStorage and sessionStorage is that sessionStorage is deleted after closing the browser while localStorage remains after closing the browser

LocalStorage

The localStorage object stores data with no expiration date. The data will not be deleted when the browser is closed, and will be available the next day, week, or year. The localStorage property is read-only.
